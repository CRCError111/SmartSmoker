# Создание тестового пользователя-администратора

## 📋 Информация о пользователе

- **Username**: `test_user`
- **Email**: `test_user@smartsmoker.local`
- **Пароль**: `Test_User_12!@`
- **Роль**: `admin` (администратор)
- **Статус**: Активен и подтверждён

## 🚀 Быстрый старт

### Вариант 1: Через phpMyAdmin

1. Откройте phpMyAdmin
2. Выберите базу данных `smart_smoker_db`
3. Перейдите во вкладку "SQL"
4. Скопируйте содержимое файла `create_test_admin_user.sql`
5. Вставьте в поле SQL-запроса
6. Нажмите "Выполнить"

### Вариант 2: Через командную строку MySQL

```bash
mysql -u root -p smart_smoker_db < database/create_test_admin_user.sql
```

### Вариант 3: Через PHP-скрипт

Если нужно сгенерировать новый хеш пароля:

```bash
php database/generate_password_hash.php
```

Затем скопируйте полученный хеш и обновите SQL-скрипт.

## 🛠️ Устранение проблем

### Ошибка: "Username уже существует"

Если пользователь уже существует, удалите его:

```sql
DELETE FROM users WHERE username = 'test_user' OR email = 'test_user@smartsmoker.local';
```

Затем повторно выполните скрипт создания.

### Ошибка: "Неверный пароль"

Если пароль не работает, сгенерируйте новый хеш:

```bash
php database/generate_password_hash.php
```

Затем обновите пароль в базе данных:

```sql
UPDATE users 
SET password_hash = 'ВАШ_НОВЫЙ_ХЕШ' 
WHERE username = 'test_user';
```

## 📝 Примечания

- Хеш пароля генерируется с помощью `password_hash()` в PHP
- Используется алгоритм bcrypt (PASSWORD_DEFAULT)
- Пароль соответствует требованиям безопасности:
  - Минимум 8 символов
  - Содержит заглавные и строчные буквы
  - Содержит цифры
  - Содержит специальные символы

## 🔒 Безопасность

⚠️ **ВАЖНО**: Этот пользователь предназначен только для тестирования!

После завершения тестирования:
1. Удалите этого пользователя
2. Или измените пароль на более безопасный
3. Не используйте этот пароль в продакшене

```sql
-- Удалить тестового пользователя
DELETE FROM users WHERE username = 'test_user';

-- Или изменить пароль
UPDATE users 
SET password_hash = password_hash('НОВЫЙ_БЕЗОПАСНЫЙ_ПАРОЛЬ', PASSWORD_DEFAULT) 
WHERE username = 'test_user';
```
